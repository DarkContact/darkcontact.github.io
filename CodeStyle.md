## Naming ##

### File naming ###
```c++
FilenameForClass.h
FilenameForClass.cpp
    
// For Wrapper:
SDL2_H.h
Optional_H.h
```

### Variable naming ###
```c++
int code = 0;
int longCode = 1;
float light = 1.5f;
double veryLongCode = 1.0;
std::string filename = "data/test.png"; // fileName ok, but better one entity, one word
```
